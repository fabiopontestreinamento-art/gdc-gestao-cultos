✝ Criar Super Admin

Primeiro acesso! Siga os passos no Supabase para criar o administrador principal:

  1. Acesse supabase.com → seu projeto GDC
  2. Menu esquerdo → Authentication → Users
  3. Clique em "Add user" → "Create new user"
  4. Email: superadmin@gdc.app
    Senha: escolha uma senha forte
  5. Clique em "Create user" e copie o UUID gerado
  6. Vá em Table Editor → tabela perfis → Insert row
  7. Preencha: id = UUID copiado, nome_completo = Super Admin, telefone = superadmin, cargo = super_admin, is_admin = true
  8. Salve. Depois faça login com o email superadmin@gdc.app e sua senha
Ok!
